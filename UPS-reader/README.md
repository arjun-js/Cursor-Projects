# UPS Reader

Monorepo:
- `client/`: React (Vite)
- `server/`: Express API (proxy for UPS hosts; to be implemented)

## Setup

```bash
cd UPS-reader
npm run install:all
```

## Dev

```bash
cd UPS-reader
npm run dev
```

Server: `http://localhost:3001/api/health`
Server: `http://localhost:8800/api/health`
